# SwimRecords Backend

Backend API voor de SwimRecords applicatie. Deze service beheert zwemresultaten, zwemmers en wedstrijden en levert data aan de frontend.

---

## Features

- Beheer van zwemresultaten  
- CRUD operaties voor zwemmers    
- REST API voor frontend communicatie  
- Database-integratie  

---

## Tech Stack

- **Runtime:** Node.js  
- **Framework:** Express  
- **Database:** (bijv. MongoDB / PostgreSQL)  
- **ORM/ODM:** (bijv. Prisma / Mongoose)  
- **API:** REST  

---

## Installatie

Volg deze stappen om de backend lokaal te draaien:

# Clone de repository
git clone https://github.com/mauddevisser/swimrecords-backend.git

# Ga naar de projectmap
cd swimrecords-backend

# Installeer dependencies
npm install

# Start de server
npm run dev
